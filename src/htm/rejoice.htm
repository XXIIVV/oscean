Rejoice!

Rejoice is a concatenative programming language where data is encoded as multisets and compose by multiplication. Think Fractran, without the rule-searching of rewriting, or a strange Forth in which the program state is unordered.

Basics

The program state is a bag of unordered things, transformations are represented as fractions where the denominator indicates things to remove from the bag, and the numerator, things to add.

( a comment )

red green blue  yellow/[green blue]
red yellow

Evaluation consists of applying fractions from left to right, each to exhaustion, until the bag stabilizes.

blue^3        black/pink white/blue red/white^3
blue^3        white/blue red/white^3
blue^2 white  white/blue red/white^3
blue white^2  white/blue red/white^3
white^3       red/white^3
red

A fraction can put another fraction in the bag.

x  [y z/y]/x 
y  z/y
z

Logic

Rejoice For example here is a NOT logic gate, notice the a resource.

false not  true/[false not] false/[true not]
true       false/[true not]
true

true not  true/[false not] false/[true not]
true not  false/[true not]
false

Building on this idea, we can make a OR gate.

x y or  true/[x y or] true/[x or] true/[y or] false/or
true    true/[x or] true/[y or] false/or
true    true/[y or] false/or
true    false/or
true

or     true/[x y or] true/[x or] true/[y or] false/or
or     true/[x or] true/[y or] false/or
or     true/[y or] false/or
or     false/or
false

Arithmetic

The fraction x/y drains y into x, effectively finding the sum by resource transfer. The sum:

x^2 y^3  x/y
x^3 y^2  x/y
x^4 y    x/y
x^5

The fraction 1/(x y) finds the difference between two symbols, but the program must handle positive and negative results:

x^4 y^2  1/[x y] pos/x neg/y
x^3 y    1/[x y] pos/x neg/y
x^2      1/[x y] pos/x neg/y
x^2      pos/x neg/y
x pos    pos/x neg/y
pos^2    neg/y
pos^2

x^2 y^4  1/[x y] pos/x neg/y
x y^3    1/[x y] pos/x neg/y
y^2      1/[x y] pos/x neg/y
y^2      pos/x neg/y
y^2      neg/y
y neg    neg/y
neg^2    neg/y
neg^2

To double the value of a symbol:

x^3      r^2/x
x^2 r^2  r^2/x
x r^4    r^2/x
r^6      r^2/x
r^6

To divide by three:

x^7        res/x^3 rem/x
x^4 res    res/x^3 rem/x
x res^2    res/x^3 rem/x
x res^2    rem/x
res^2 rem

Functions

Functions are symbols to be replaced by the body of their definition. A definition starts with colon, and ends with a semi-colon. The definition comment indicates the state of the bag before and after the transformation.

: And? ( x y -- bool ) a true/[a x y] false/[a x] false/[a y] false/a ;

x y And?
x y a     true/[a x y] false/[a x] false/[a y] false/a 
true      false/[a x] false/[a y] false/a 
true      false/[a y] false/a 
true      false/a 
true

Loops

A function can include its own name in the definition, allowing for loops:

: loop ( x -- ) x done/x^4 [x^2 loop]/x ;
x loop
x         done/x^4 [x^2 loop]/x
x         [x^2 loop]/x
x^2 loop
x^2       done/x^4 [x^2 loop]/x
x^2       [x^2 loop]/x
x^3 loop
x^3       done/x^4 [x^2 loop]/x
x^3       [x^2 loop]/x
x^4 loop
x^4       done/x^4 [x^2 loop]/x
done      [x^2 loop]/x
done
