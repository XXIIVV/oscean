Uxntal Strings

Uxntal has superficial support to convert raw ascii characters into bytes. An ascii token begins with the double-quote character, there is no escaping. For example, a linebreak and null-terminated "Hello world" looks like:

"Hello 20 "world. 0a $1

The same token type can be used to inline characters, for example, the following example will push the hexadecimal value 0x61 on the stack before printing the letter "a".

LIT "a .Console/write DEO
