Donsol Famicom is a port of the dungeon-crawler card game for the classic Nintendo.

Donsol is a dungeon crawler card game played with a standard deck of 54 playing cards. The game was programmed in 6502 assembly, the art assets were created with Nasu by Rekka, the game was designed by John Eternal and the famicom rom is released by Hundred Rabbits.

You can find the rules to the game on the main Donsol page. The Donsol assembly source code follows the lin6 styleguide.
