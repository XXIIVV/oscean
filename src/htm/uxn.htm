Uxn is the virtual machine powering the Hundred Rabbits software.

The virtual machine, programmable in Uxntal, is a portability layer with a focus on hosting graphical tools and games. It lives at the heart of the Varvara computer system and this wiki.

  • Uxn Instructions: Reference, Tests
  • Varvara Devices: Reference, Tests
  • Emulators: CLI, GUI(X11)

A small and habitable environment for personal computing that allows to comfortably write programs in assembly to tackle general computation projects with what feels like high-level abstractions!
