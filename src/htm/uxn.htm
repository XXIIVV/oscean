Uxn is the virtual machine living at the heart of the Hundred Rabbits ecosystem.

The virtual machine, programmable in Uxntal, is a portability layer and an attempt at crafting a comfortable amber for personal computing that is capable of hosting a variety of small interactive graphical software. It is powering this wiki.

Quick Setup

To get started, equip yourself with an assembler to convert tal source files into rom binary files, and an emulator to evaluate the rom files. The system below includes the Console device:

# Build an emulator
cc uxnmin.c -o uxnmin

# Build an assembler
cat drifloon.rom.txt | uxnmin xh.txt > drifloon.rom

# Assemble a tal file
cat opctest.tal | uxnmin drifloon.rom > opctest.rom

# Run a rom file
uxnmin opctest.rom

Summary

Uxn

	A byte' is 8 bits
	A short" is two big-endian bytes

	There are 2 stacks
	Each stack has 256 bytes of data
	Each stack pointer is a byte and wraps

	Dev(D) is 256 bytes and wraps
	Ram(M) is 65536 bytes and wraps
	Rom is loaded and evaluated at 256

opcode<-M[pc], pc<-pc+1

	20|opcode Values are shorts instead of bytes, addr is abs" or rel' 
	40|opcode Values are read or written on . the other stack
	80|opcode Values are not consumed when read
