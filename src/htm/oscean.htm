Oscean is a wiki engine.

Oscean is written entirely in assembly, doing away with any dependencies beyond the thin emulation layer of the Uxn virtual machine. The assembler itself is written in the language it assembles and is bootstrapped from a minuscule binary seed.

It is designed to be deployed from low-power devices. The generated files are optimized for screen-readers and terminal browsers, using no javascript, and a stylesheet of a mere 30 lines. The wiki and its related tools are generally used offline, so there are no offsite queries for critical resources. Since 2006, this ecosystem has grown into a collection of tools which also includes a time tracker. The journal entries are catalogued in the Arvelie time format.
