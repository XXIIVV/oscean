Uxn is the virtual machine powering the Hundred Rabbits software.

The virtual machine, programmable in Uxntal, is a portability layer with a focus on hosting graphical tools and games. It lives at the heart of the Varvara computer system and this wiki.

Setup

Here is a quick way to get started writing programs with a little self-hosted assembler and an emulator which includes the Console device:

# Build emulator
cc uxnmin.c -o uxnmin

# Build assembler
xxd -r -p drifloon.rom uxnasm.rom

# Assemble source
echo "#1234 #010e DEO" | uxnmin uxnasm.rom > result.rom

# Run program
uxnmin result.rom

Stack-machines make for an habitable abstraction for personal computing that allows one to comfortably write programs with assembly to tackle general computation projects with what feels like high-level abstractions!
