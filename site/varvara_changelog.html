    Varvara Changelog

    79K: In the File Device, the character spacer, between the file detail and file name in the directory view, has changed from a space(0x20) to a tab(0x09). This change helps with the parsing of the directory view, especially for files with spaces in their names, and make the view a natural string for tsv parsers, like Nebu.
