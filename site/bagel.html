    Gödel's bag, Gödel's bag,
    Gödel's bag.

    Bägel is a programming language where data is encoded as multisets(unordered bags of things) and compose by multiplication. Think Fractran, without the rule-searching of rewriting, or a strange Lisp in which cons cells are unordered lists.

    In this mirror world, order does not matter, only presence and reduction.

    Basics

    Parentheses denotes a bag, which may contain whole numbers, fractions or other bags. The order of items in a bag does not matter.

    ; This is a comment

()          ; Empty(1), identity
(2)         ; 2
(2 (3))     ; 6
(3 (2 5/2)) ; 15

    The evaluation consists of applying fractions on whole numbers starting at the deepest nested level upward until the bag stabilizes. The bag left after the last application is the output of a program.

    (3/2 2)       ; (3/2 2) -> 3
(3/5 125)     ; (1/5 5^3) -> (1/5 3 5^2)
(1/5 (5/7 7)) ; (1/5 (5/7 7)) -> (1/5 5) -> ()

    Fractions do not fall through to the outer bag if application fails.

    ((3/4 2) 2) ; The inner bag cannot be reduced.
4           ; The fraction 3/4 is discarded.

    Fractions may output other fractions.

    (2/3/5 3 5)   ; (2/3/5 3 5) -> (2/3 3) -> 2
((5/3 3)/7 7) ; ((5/3 3)/7 7) -> (5/3 3) -> 5
(3/2/2/2 2^4) ; (3/2/2/2 2^4) -> (3/2/2 2^3) -> (3/2 2^2) -> 6

    When a bag contains multiple fractions, fractions are composed before application.

    (2/3 2/5 3 5) ; Two fractions in the bag
(4/15 3 5)    ; The two fractions are composed before application
4             ; Application

    Non-numeric symbols are assigned an unused prime factor in the program.

    (red green blue)          ; A bag with 3 colors
((black white)/red red^2) ; ((black white)/red red^2) -> (red black white)
(fizz/2^3 2 2 2)          ; (fizz/2^3 2 2 2) -> fizz

    Quotations

    A quoted fraction is kept after application, this is the mechanism through which recursion is made.

    ('5/3 3^4)     ; Drain 3 into 5
('5/3 3^3 5)
('5/3 3^2 5^2)
('5/3 3 5^3)
('5/3 5^4)
5^4

    Quoted fractions also compose, non-quoted fractions will compose with quoted fraction. The resulting fraction will be a quotation.

    ('2/3 '2/5 3^2 5^2) ; Two quoted fractions in the bag.
('4/15 3^2 5^2)     ; The two fractions are composed before application
('4/15 2^2 3 5)
('4/15 2^4)
2^4

    Logic

    Logic is also done by the application of fractions.

    (false/x (false/y (true/(x y) x y)))
(false/x (false/y true))
(false/x true)
true

    (false/x (false/y (true/(x y) y)))
(false/x (false/y y))
(false/x false)
false

    Arithmetic

    The fraction 'y/x drains x into y, effectively finding the sum:

    ('y/x x^3 y^2)
('y/x x^2 y^3)
('y/x x y^4)
('y/x y^5)
y^5

    The fraction '1/(x y) finds the difference, and the outer bag moves the result into a third symbol:

    (result/x ('1/(x y) x^4 y^2))
(result/x ('1/(x y) x^3 y))
(result/x ('1/(x y) x^2))
(result/x ('1/(x y) x^2))
(result/x x)
result
